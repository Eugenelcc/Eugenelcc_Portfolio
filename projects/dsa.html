Full Stack Development - NodeJS

Background

This is a solo Data structure and algorithm project work which split into 2 different code projects. The first code requires me to make a Inventory's management application using 12 weeks of using Python and it's data structure algoritm. The second code is using a list of given sequence positive integer and determine if there exist two integers in the sorted sequence.

Learning about different type of sorting algorithms and display information into the application. Such algorithms use in this application for sorting and arranging are:

Bubble Sort Algorithms: Swapping or doing multiple passes to the string of numbers or other elements in a correct order.

Insertion Sort Algorithms: Maintaing a collection of sorted items and inserting unsorted data into sorted data just like a playing cards.

Quick Sort Algorithms: Using divide and conqurer strategy by picking a random element as pivot and sort the rest accordingly.

1st project: Ezy's Shop Inventory management

In this project, I decide to go for food-related stocks inventory management, where shops owners or managers can use this application to manage their inventory stocks easily, this also comes with an additonal features where backups and be export and import by using CSV file.

The code uses Object-oriented programming(OOP) model or concept to orgranise the data base on (Item's name, Item's descriptions, Item's selling price, Item's stock level and Item's manufacturer). Users using this application are given the rights to add new products, delete,view and update using the CRUD operations and data will be store in .dat file format by using Python's Shelves

Sorting of products based on Selling price and the number of stocks available for shop owners to view, this is the use of different type of sorting algorthms where Insertion sort is use to sort the selling price and bubble sort is use to sort the number of inventory stocks.

Shop owners can also export their data into a backup-file data into a CSV excel file, if in the case of file corruption in the system. By importing the backup data to the application, the application will prompt and give a warning to the user that the current data in the shelve storage will be overwrite by the backup data.

2nd project: SEQ sequence project

The SEQ sequence project was given as a challenge question for additonal points, how the system works is there is a fixed positive integer prepared. The user's input a random number which represent as "Z" while the task is to loop through the sequence of number and select 2 numbers (X,Y) from the list which will be added up and match the added up number with "Z"

By using Quick Sort algorithm, using the divide and conqurer strategy, first it will arrange the number into ascending order and this will pick X as the fist number and Y as the last number for 2 pivot and compare the sequence number which then will determine if both (X+Y) = Z as the final results.
